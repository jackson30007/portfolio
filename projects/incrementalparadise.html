Incremental Paradise is a standalone server. The game is unique, its a mix between a incremental & mmorpg. The main game loop involves completing sets of tasks to level up. After gaining enough levels you can prestige, after gaining enough prestiges you can ascend, etc. There are currently 4 reset layers in the game, each new reset layer unlocks a new world. Each world has unique mobs, crops, ores, trees, fish & bosses. Lots of crops and some trees have unique harveting methods that act like little minigames. Players can also unlock a ton of unique active abilities to make harvesting resources feel fun and rewarding.

The game supports multiple servers. All player data, including inventories & storage menus are persistent across multiple servers. There is even support for a player to have multiple inventories, as some worlds have unique inventories. When a server is going to restart players will be transfered to another server instead of being kicked from the server altogether.

Incremental Paradise has been my most fun project. After working with Minecraft 1.8 for years, working with the latest versions of the game has been awesome. All the new features added to the game bring about so many possibilities for what can be done in vanilla Minecraft. For example, display entities. Having these added to the game was a blast. One of my favorite things I did with display entities is making a giant boss model. Complete with animations and no resource pack required.
